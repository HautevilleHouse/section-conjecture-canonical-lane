# EG1 Public Note

The `EG1` package establishes the positive projected response floor.

Define the projected response operator `E_tau = Pi_resp L_tau Pi_resp` on the admissible response sector.

Raw theorem constant:

`kappa_section^(raw) := c_section_raw * section_density_raw - e_section_raw`.

Closure criterion:

- the projected response is finite and theorem-level,
- `kappa_section > 0` on the declared admissible tube.

Main interpretation: the admissible closure cannot collapse because the projected section response remains strictly positive.
